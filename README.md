# Task Manager

A full-stack Task Manager application built with the **MERN stack**.  
Users can create tasks, mark them as completed, and manage their work efficiently.

## Features
- User authentication
- Create, update, and list tasks
- Mark tasks as completed / undo
- Persistent storage with MongoDB

## Tech Stack
- **Frontend:** React, Axios
- **Backend:** Node.js, Express
- **Database:** MongoDB
